## 五层能力矩阵

### 各层能力与依赖阶段

| 层级 | 能力 | 依赖阶段 |
|------|------|----------|
| **感知层** | 文本对话 REST API | 阶段 0-3 | 
| | 文件导入 RAG | 阶段 2 |
| | 定时触发 cron | - | 规划中 |
| **推理层** | 文本生成 Ollama | 阶段 1 |
| | 语义 Embedding bge-m3 | 阶段 1 |
| | 工具调用 | 阶段 3 |
| | 多模型路由 | 阶段 6 |
| **记忆层** | 短期对话历史 | 阶段 3 |
| | 长期 Qdrant 向量记忆 | 阶段 4 |
| | 用户画像 SQLite | 阶段 4 |
| **执行层** | 文件操作 Agent 工具 | 阶段 3 |
| | Shell 安全沙箱 | 阶段 3 |
| | RAG 知识检索 | 阶段 2-3 |
| **调度层** | 模型路由器 | 阶段 6 |
| | Nginx 负载均衡 | 阶段 0/5 |
| | 任务队列 Celery | - |
